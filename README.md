# vote-with-your-feet

A Python project initialized with UV and pytest-describe.

## Setup

This project uses [UV](https://github.com/astral-sh/uv) for dependency management.

```bash
uv sync
```

## Running Tests

Run tests with pytest-describe:

```bash
uv run pytest
```

For verbose output:

```bash
uv run pytest -v
```

## Running the Application

```bash
uv run python hello.py
```
